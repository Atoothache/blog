# python模块安装

1、参照网上的安装方法通过pip来安装，未成功

​ pip install 包名

2、从github上下载urllib3

​ https://pypi.org/ 官网查找需要的离线包

​ 解压离线包，进入对相应目录执行 python setup.py install

​ python setup.py install

​ 还需要将对应文件夹下包名为对应的包名（如安装requests包则是文件夹下requests目录）复制到Lib下，否则在pycharm下不可用
